Example 1
{: .label .label-yellow }

```python
PLOT_SETUP = {
    'NAME': 'Treemap Chart',
    'WIDTH': 30,
    'HEIGHT': 15,
    'DPI':6000,
    'EXTENSION':'svg'
}

DATASET = {
    'VALUES': [250, 120, 280, 320, 140, 95],
    'COLORS': ['#91DCEA', '#64CDCC', '#5FBB68', '#F9D23C', '#F9A729', '#FD6F30'],
    'LABELS': ['Group 1', 'Group 2', 'Group 3', 'Group 4', 'Group 5', 'Group 6']
}


TREEMAP_CHART(DATASET=DATASET, PLOT_SETUP=PLOT_SETUP)
```
