<p align = "justify">The function displays the plot on the screen and saves it to the local folder of the <code>.ipynb</code> / <code>.py</code> file.</p>


Example 1
{: .label .label-blue }

<p align = "justify">We use the <code>HEATMAP_CHART</code> function to plot a week food consumption.</p>

```python
# Data
DF = {
    'VALUES': [250, 120, 280, 320, 140, 95], 
}

# Chart setup
PLOT_SETUP = {
    'NAME': 'figure1-8-1',
    'WIDTH': 30,
    'HEIGHT': 15,
    'COLORS': ['#91DCEA', '#64CDCC', '#5FBB68', '#F9D23C', '#F9A729', '#FD6F30'],
    'LABELS': ['Group 1', 'Group 2', 'Group 3', 'Group 4', 'Group 5', 'Group 6'],
    'LABEL SIZE': 20,
    'EXTENSION':'svg',
    'DPI':600
}

# Data statement 
DATA = {'DATASET': DF}

# Call function
TREEMAP_CHART(DATASET = DATA, PLOT_SETUP = PLOT_SETUP)
```
