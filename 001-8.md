<h2>Example 1</h2>

```python
from EASYPLOT_TOOLBOX import *

config = {
    'NAME': 'Join Histogram Chart',
    'WIDTH': 20,
    'HEIGHT': 10,
    'DPI':600,
    'EXTENSION':'svg',
    'K': 500
}

dataset = {
    'LABELS': ["G1", "G2", "G3", "G4", "G5"],
    'VALUES': [40, 50, 140, 50, 40],
    'COLORS': ["#FDAE61", "#FEE08B", "#FFFFBF", "#E6F598", "#ABDDA4"]
}


JOIN_HIST_CHART(CONFIG=config, DATASET=dataset)
```
