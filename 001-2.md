Example 1
{: .label .label-blue }

<p align = "justify">We use the <code>LINE_CHART</code> function to plot a series of banana prices.</p>

```python
# Data
DF =  pd.DataFrame({'x0': [2012, 2013, 2014, 2015, 2016],
                    'y0': [2.5, 3.0, 2.2, 4.5, 3.5]
                   })

# Chart setup
CHART_CONFIG = {
              'NAME': 'figure5',
              'WIDTH': 15., 
              'HEIGHT': 7.5,
              'MARKER': ['s'],
              'MARKER SIZE': 3,
              'LINE WIDTH': 4,
              'LINE STYLE': ['--'],
              'X AXIS LABEL': 'Year',
              'X AXIS SIZE': 10,
              'Y AXIS LABEL': 'Price ($\$$)',
              'Y AXIS SIZE': 10,
              'AXISES COLOR': '#000000',
              'LABELS SIZE': 14,
              'LABELS COLOR': '#000000',
              'CHART COLOR': ['#000000'],
              'ON GRID?': True,
              'LEGEND': ['banana'], # or without legend 'LEGEND': [None]
              'LOC LEGEND': 'upper left',
              'SIZE LEGEND': 12,
              'Y LOG': False,
              'X LOG': False,
              'DPI': 200, 
              'EXTENSION': 'svg'
             }

# Data statement 
DATA = {'DATASET': DF}

# Call function
LINE_CHART(DATASET = DATA, PLOT_SETUP = CHART_CONFIG)
```

Example 2
{: .label .label-yellow }

<p align = "justify">We use the <code>LINE_CHART</code> function to plot a series prices of three fruits.</p>

```python
# Data
DF =  pd.DataFrame({'x0': [2012, 2013, 2014, 2015, 2016],
                    'y0': [2.5, 3.0, 2.2, 4.5, 3.5],
                    'x1': [2012, 2013, 2014, 2015, 2016],
                    'y1': [2.2, 2.8, 2.4, 4.3, 3.2],
                    'x2': [2012, 2013, 2014, 2015, 2016],
                    'y2': [2.6, 2.9, 2.3, 4.2, 3.6]
                   })

# Chart setup
CHART_CONFIG = {
              'NAME': 'figure1-2-2',
              'WIDTH': 15.0, 
              'HEIGHT': 7.5,
              'MARKER': ['1', 's', 'o'],
              'MARKER SIZE': 3,
              'LINE WIDTH': 4,
              'LINE STYLE': ['--', '-', ':'],
              'X AXIS LABEL': 'Year',
              'X AXIS SIZE': 10,
              'Y AXIS LABEL': 'Price ($\$$)',
              'Y AXIS SIZE': 10,
              'AXISES COLOR': '#000000',
              'LABELS SIZE': 14,
              'LABELS COLOR': '#000000',
              'CHART COLOR': ['#000000','olive','r'],
              'ON GRID?': True,
              'LEGEND': ['Banana', 'Apple', 'Orange'],
              'LOC LEGEND': 'upper left',
              'SIZE LEGEND': 8,
              'Y LOG': False,
              'X LOG': False,
              'DPI': 600, 
              'EXTENSION': 'svg'
             }

# Data statement 
DATA = {'DATASET': DF}

# Call function
LINE_CHART(DATASET = DATA, PLOT_SETUP = CHART_CONFIG)
```

<p align = "center"><b>Figure 2.</b> Prices series between 2012 to 2016.</p>
